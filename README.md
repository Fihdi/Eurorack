# Open Source Eurorack Synthesizer

Schematics and PCBs for my Eurorack modules

**2nd May 2026: UNDER CONSTRUCTION!** I am currently fixing and re-organizing every module.

The following modules have be re-worked and can be found in their own repositories:

* 30hp Drum, Hi Hat and WAV Player: [MiniDrumkit](https://github.com/Fihdi/MiniDrumkit)
* 8hp ADSR Envelope Generator: [SCULPT](https://github.com/Fihdi/SCULPT)
* 8hp Dual VCAs / Ringmodulators: [BIPO](https://github.com/Fihdi/BIPO)
* 8hp 12dB/Oct state variable VCF: [SVF12](https://github.com/Fihdi/SVF12)
* 8hp Voltage Controlled Attack/Release and Slew Rate limiter: [VCAR](https://github.com/Fihdi/VCAR)
